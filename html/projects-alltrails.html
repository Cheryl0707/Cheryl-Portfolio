AllTrails (Add-on Features)

“Handbook” feature is integrated many outdoor knowledges and hiking “stuffs you should know”. It is friendly for the beginner, and also welcome to the pro hikers to share their experience.

- Project Type: UI + UX Design

- Roles: Solo Project

- Date: Oct - Nov 2022

- Tools: Figma / Protopie

Explore AllTrails

Primary Purpose

AllTrails can help the user who interested in hiking, biking and running to get trails information around a specific location. These trails can be filtered for different demands (dog-friendly, kid-friendly, wheelchair-friendly), or they can be filtered by distance and intensity. User can upgrade to Pro version to have more premium function (Emergency contact, offline map)

"Explore"

This feature directly addresses the demands for users to find a trail by providing their brief information.
There are some categorization:
-Top trails nearby: this category mainly recommend the featured trails to the user.
-Best views nearby: This category mainly recommend the trails with beautiful views. Categorization bar: select activity type for more specific recommendation.

"Navigate"

This feature can navigate user in the trails and record the hiking statistics. If the user hit “pause”, AllTrails will end the navigation and ask the user for the comments and photos of the trip.

Interface Design Patterns

Layout: Layout: The layout is really clear for the user. Each function logically guide the user. The key information are visually isolated.

Colours: The main colour of this app is Green, grey and white. User can naturally connect the colour palette with “nature”, “life”, “relaxation”, “healthy” etc….

Icons: The icons are visible and easy to understand.

Add On Features

"Handbook"

“Handbook” function is integrated many outdoor knowledges and hiking “stuffs you should know”. It is friendly for the beginner, and also welcome to the pro hikers to share their experience.

Content

Hiking safety tips / Seasonal hiking hotspot/ Packing tips / Hiking Essentials / First aid guideline

Goal

- Build a platform to popularize hiking knowledge.
- Enhance the connection between users and platforms, novices and masters.
- Eliminate knowledge barriers.

Why?

- Providing users with information can increase their understanding of outdoor sports.
- In case of a dangerous situation, it is necessary to master the correct method of self-rescue

"Distance Rewards"

“Distance Rewards” feature is encourage the user to hike. If the user reached certain distance, they will get AllTrails Premium on trial. With this version they can explore more of the app.

Content

There is a data collect cycle to guide users to reach certain amount of miles to get rewards.

Goal

The goal is to encourage users to hiking, running and cycling. Provide a window for user to try the Premium version or plant more trees.

Why?

Because many important function of AllTrails are Premium only. We can provide a way to popularize hiking and break the paywall between platform and user.
