AllTrails (Add-on Features)

“Handbook” feature is integrated many outdoor knowledges and hiking “stuffs you should know”. It is friendly for the beginner, and also welcome to the pro hikers to share their experience.

- Project Type: UI + UX Design

- Roles: Solo Project

- Date: Oct - Nov 2022

- Tools: Figma / Protopie

Explore AllTrails

Primary Purpose

AllTrails can help the user who interested in hiking, biking and running to get trails information around a specific location. These trails can be filtered for different demands (dog-friendly, kid-friendly, wheelchair-friendly), or they can be filtered by distance and intensity. User can upgrade to Pro version to have more premium function (Emergency contact, offline map)

"Explore"

This feature directly addresses the demands for users to find a trail by providing their brief information.
There are some categorization:
-Top trails nearby: this category mainly recommend the featured trails to the user.
-Best views nearby: This category mainly recommend the trails with beautiful views. Categorization bar: select activity type for more specific recommendation.

"Navigate"

This feature can navigate user in the trails and record the hiking statistics. If the user hit “pause”, AllTrails will end the navigation and ask the user for the comments and photos of the trip.
